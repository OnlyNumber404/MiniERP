# 💰 MINI ERP
A mini-ERP system designed to help users track transactions and manage income and expenses efficiently.

## 🚀 About The Project

This is a **Mini ERP project** created to help users manage their transactions and track income and expenses efficiently.

The purpose of this mini-ERP is to help users manage their transactions and track their income and expenses. It features an analytics tool to evaluate assets, presented through a visually appealing and dynamic dashboard built with Laravel. Additionally, the system supports user inputs for images, monetary amounts, and textual descriptions, while seamlessly generating comprehensive income and expense reports.

## 🔥 API Integration
This project uses the **[CoinGecko](https://www.coingecko.com/en/api)** to retrieve cryptocurrency data.

**Key Features:**
* 📊 **Analytics Dashboard:** Visualize your assets and track income and expenses.
* 📚 **Financial Reporting:** Simplifies cash flow management by generating detailed income and expense reports.
* 📝 **Rich data inputs:** Supports user uploads and inputs, including images, transaction amounts, and detailed text descriptions.
* 👤 **User Authentication:** Allows users to register and log in to access their financial data.
* 🔍 **Cryptocurrency Search:** Check prices and details for any cryptocurrency.
* ☁️ **API Integration:** Fetches real-time data from a CoinGecko API.

## 🛠️ Built With

* **[Laravel](https://laravel.com/)** - For building the user interface.
* **[Tailwind CSS](https://tailwindcss.com/)** - For styling and responsive design.
* **[Vite](https://vitejs.dev/)** - For fast development and building.
* **[Chart.js](https://www.chartjs.org/)** - For data visualization.
* **[maatwebsite Excel](https://docs.laravel-excel.com/3.1/getting-started/)** - For exporting financial data to Excel.

## 💻 Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/OnlyNumber404/MiniERP.git
    ```
2.  **Install dependencies**
    ```bash
    composer install
    ```
3. **Install Frontend Dependencies**
    ```bash
    npm install
    ```
4. **Generate Application Key**
    ```bash
    php artisan key:generate
    ```
5.  **Run the project**
    ```bash
    composer run dev
    ```
